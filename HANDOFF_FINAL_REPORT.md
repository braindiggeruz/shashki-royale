## Команды для продолжения работы

```bash
# Установка зависимостей
cd /home/ubuntu/shashki-royal
pnpm install

# Пересборка веб-приложения
pnpm build

# Синхронизация Capacitor
npx cap sync

# Пересборка Android (если Java доступна)
cd android
./gradlew clean
./gradlew assembleDebug

# Установка APK на устройство
adb install -r delivery/shashki-royal-online-debug.apk
```
