# Шашки Рояль: онлайн-комнаты без паролей и Android APK

Готова реализация простого онлайн-входа через **комнаты без паролей**. В проект добавлены сценарии создания комнаты, входа по короткому коду, входа по ссылке-приглашению, копирования ссылки и системного шаринга на мобильных устройствах. Пользователю не нужно вводить пароль: достаточно открыть ссылку или вставить код комнаты.

## Что реализовано

| Блок | Результат |
|---|---|
| Онлайн-комнаты | Создание комнаты с коротким кодом и ссылкой-приглашением. |
| Вход без пароля | Игрок входит по коду/ссылке, без email и пароля. |
| Мобильный UX | Добавлены копирование ссылки и `navigator.share`, где поддерживается устройством. |
| Сервис комнат | Обновлён `src/services/gameRooms.ts`: генерация кодов, нормализация кода, поиск комнаты по invite-коду, повторная генерация при конфликте. |
| Лобби | Обновлён `src/pages/Lobby.tsx`: быстрые сценарии «создать комнату» и «войти по коду/ссылке». |
| Android | Добавлена Capacitor-обёртка и Android-проект. Доступный debug APK приложен отдельно. |

## Важное по APK

В папку выдачи добавлен файл `shashki-royal-online-debug.apk`. Во время последней пересборки Android возник блокер окружения SDK: локальный `android-36/android.jar` не загрузился корректно, из-за чего последняя пересборка остановилась на `app:processDebugResources`. Поэтому я также передаю полный исправленный проект, чтобы при наличии нормального Android SDK 36 APK можно было пересобрать командой:

```bash
cd shashki-royal
pnpm build
npx cap sync android
cd android
./gradlew assembleDebug
```
