## 1. Выявленные проблемы и их решение

### 1.1 Критическая проблема: APK не устанавливается

**Проблема:** Ошибка "Приложение не установлено" при попытке установки на Android-устройство.

**Причина:** APK был переупакован (unzip → zip) для обновления web-assets, но не был заново подписан. Android 7+ требует валидную подпись (APK Signature Scheme v2/v3). При изменении содержимого архива старая подпись становится недействительной.

**Решение:**
- Установлены утилиты `zipalign` и `apksigner`
- Создан debug keystore (`/home/ubuntu/debug.keystore`)
- APK выровнен с помощью `zipalign -p -f 4`
- APK подписан с помощью `apksigner sign`
- Подпись проверена с помощью `apksigner verify` — ✅ валидна

**Статус:** ✅ **ИСПРАВЛЕНО**

---

### 1.2 Ошибка в конфигурации Supabase URL

**Проблема:** В файле `.env` была опечатка в URL Supabase: `jsykbnkbwwsxcdurzc` вместо `jsykbnkbrwwsxcdurzcw`.

**Решение:**
- Исправлена строка в `.env`:
  ```
  VITE_SUPABASE_URL=https://jsykbnkbrwwsxcdurzcw.supabase.co
  ```
- Пересобран веб-билд (`pnpm build`)
- Синхронизирован Capacitor (`npx cap sync android`)
- Обновлён APK с исправленной конфигурацией

**Статус:** ✅ **ИСПРАВЛЕНО**

## 7. Возможные проблемы и их решение

### 7.1 Google OAuth не работает на мобильном устройстве

**Решение:**
1. Перейти в [Supabase Console](https://app.supabase.com)
2. Выбрать проект
3. Перейти в **Authentication → Providers → Google**
4. Убедиться, что указаны:
   - Client ID (из Google Cloud Console)
   - Client Secret (из Google Cloud Console)
   - Redirect URLs включают `com.shashki.royal://login-callback`
5. Сохранить изменения

### 7.2 Приложение не подключается к Supabase

**Решение:**
1. Проверить интернет-соединение на устройстве
2. Убедиться, что Supabase сервер доступен
3. Проверить логи: `adb logcat | grep shashki`

### 7.3 Ошибка при установке APK

**Решение:**
1. Убедиться, что на устройстве включена опция "Установка из неизвестных источников"
2. Убедиться, что достаточно свободного места на устройстве
3. Попробовать удалить старую версию: `adb uninstall com.shashki.royal`
